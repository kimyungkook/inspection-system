# AI 주식 매매 가이드 앱 — 시작 가이드

## 처음 시작하는 분을 위한 단계별 안내

---

## STEP 1. Docker Desktop 설치

1. 구글에서 "Docker Desktop 다운로드" 검색
2. Windows 버전 다운로드 클릭
3. 설치 파일 실행 → [다음] → [다음] → [완료]
4. 컴퓨터 재시작
5. 바탕화면에 Docker Desktop 아이콘 생성 확인

---

## STEP 2. API 키 발급

### 한국투자증권 API 키
1. 한국투자증권 앱 또는 홈페이지(www.truefriend.com) 접속
2. 로그인 후 상단 메뉴 → [KIS Developers] 클릭
3. [앱 등록] → 앱 이름 아무거나 입력 (예: 내주식앱)
4. 앱키(App Key)와 앱시크릿(App Secret) 발급됨
5. 메모장에 두 값 모두 저장

### Claude API 키
1. console.anthropic.com 접속
2. 구글 계정으로 회원가입
3. [API Keys] 메뉴 → [Create Key] 클릭
4. 발급된 키를 메모장에 저장
5. 카드 등록 (사용한 만큼만 청구, 월 약 1~3만원)

---

## STEP 3. 설정 파일 만들기

1. 프로젝트 폴더 안의 `.env.example` 파일을 찾으세요
2. 그 파일을 복사해서 이름을 `.env` 로 바꾸세요
3. 메모장으로 `.env` 파일 열기
4. 아래 항목에 발급받은 키 입력:

```
KIS_APP_KEY=한국투자증권에서_받은_앱키
KIS_APP_SECRET=한국투자증권에서_받은_앱시크릿
KIS_ACCOUNT_NUMBER=계좌번호(예:50123456-01)
CLAUDE_API_KEY=Anthropic에서_받은_API키
JWT_SECRET_KEY=아무글자나_32자_이상_입력(예:MyApp_Secret_Key_2024_!!)
AES_ENCRYPTION_KEY=아무글자나_32자_입력(예:MyEncryptKey_2024_!@#$%^&*)
```

5. 저장 후 닫기

---

## STEP 4. 앱 실행

1. 키보드 **Windows키 + R** 누르기
2. `cmd` 입력 후 엔터 (검은 화면 열림)
3. 아래 명령어 입력 후 엔터:
   ```
   cd C:\프로젝트폴더경로\inspection-system
   docker-compose up -d
   ```
4. 처음에는 3~5분 정도 걸립니다 (자동 설치 중)
5. 완료 후 브라우저에서 접속:
   - API 서버: http://localhost:8000
   - API 문서: http://localhost:8000/docs
   - 작업 현황: http://localhost:5555

---

## STEP 5. 앱 중지/재시작

```
중지: docker-compose down
재시작: docker-compose up -d
```

---

## 자주 묻는 질문

**Q. 실행했는데 오류가 나요**
→ .env 파일에 입력한 값이 맞는지 확인하세요

**Q. 포트가 이미 사용 중이라고 해요**
→ Docker Desktop에서 실행 중인 컨테이너를 모두 중지 후 다시 시도

**Q. 데이터가 사라졌어요**
→ `docker-compose down -v` 는 데이터도 삭제됩니다. 중지할 때는 반드시 `docker-compose down` 만 사용하세요
