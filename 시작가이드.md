# AI 주식 매매 가이드 앱 — 시작 가이드

## 처음 시작하는 분을 위한 단계별 안내

---

## STEP 1. Docker Desktop 설치

1. 구글에서 "Docker Desktop 다운로드" 검색
2. Windows 버전 다운로드 클릭
3. 설치 파일 실행 → [다음] → [다음] → [완료]
4. 컴퓨터 재시작
5. 바탕화면에 Docker Desktop 아이콘 생성 확인

---

## STEP 2. API 키 발급

### 한국투자증권 API 키
1. 한국투자증권 앱 또는 홈페이지(www.truefriend.com) 접속
2. 로그인 후 상단 메뉴 → [KIS Developers] 클릭
3. [앱 등록] → 앱 이름 아무거나 입력 (예: 내주식앱)
4. 앱키(App Key)와 앱시크릿(App Secret) 발급됨
5. 메모장에 두 값 모두 저장

### Claude API 키
1. console.anthropic.com 접속
2. 구글 계정으로 회원가입
3. [API Keys] 메뉴 → [Create Key] 클릭
4. 발급된 키를 메모장에 저장
5. 카드 등록 (사용한 만큼만 청구, 월 약 1~3만원)

---

## STEP 3. 설정 파일 만들기

1. 프로젝트 폴더 안의 `.env.example` 파일을 찾으세요
2. 그 파일을 복사해서 이름을 `.env` 로 바꾸세요
3. 메모장으로 `.env` 파일 열기
4. 아래 항목에 발급받은 키 입력:

```
KIS_APP_KEY=한국투자증권에서_받은_앱키
KIS_APP_SECRET=한국투자증권에서_받은_앱시크릿
KIS_ACCOUNT_NUMBER=계좌번호(예:50123456-01)
CLAUDE_API_KEY=Anthropic에서_받은_API키
JWT_SECRET_KEY=아무글자나_32자_이상_입력(예:MyApp_Secret_Key_2024_!!)
AES_ENCRYPTION_KEY=아무글자나_32자_입력(예:MyEncryptKey_2024_!@#$%^&*)
```

5. 저장 후 닫기

---

## STEP 4. 앱 실행

### 방법 1 — 더블클릭으로 실행 (권장)
1. 프로젝트 폴더 안의 **앱시작.bat** 파일을 더블클릭하세요
2. 검은 창이 열리고 자동으로 서버가 시작됩니다
3. 처음에는 3~5분 정도 걸립니다 (Docker 이미지 자동 다운로드 중)
4. 완료되면 브라우저가 자동으로 열립니다:
   - API 문서: http://localhost:8000/docs
   - 백그라운드 작업 현황: http://localhost:5555

### 방법 2 — 명령어로 실행
1. 키보드 **Windows키 + R** 누르기
2. `cmd` 입력 후 엔터
3. 아래 명령어 입력:
   ```
   cd C:\프로젝트폴더경로\inspection-system
   docker-compose up -d
   ```

---

## STEP 5. 앱 중지/재시작

- **중지**: **앱중지.bat** 더블클릭 (데이터 안전하게 보존)
- **재시작**: **앱시작.bat** 다시 더블클릭

⚠️ 주의: `docker-compose down -v` 는 절대 사용 금지 — 데이터가 모두 삭제됩니다

---

## STEP 6. 텔레그램 알림 설정 (선택 — 권장)

1. 스마트폰에서 텔레그램 앱 설치 및 가입
2. 검색창에 **@BotFather** 검색 → 대화 시작
3. `/newbot` 입력 → 봇 이름 입력 (예: 내주식알림봇)
4. 발급된 **토큰**(숫자:문자 형식)을 메모장에 저장
5. 검색창에 **@userinfobot** 검색 → 대화 시작 → 메시지 아무거나 보내기
6. 내 **채팅 ID** 확인 후 메모장에 저장 (음수로 시작할 수 있음)
7. `.env` 파일 열어서 입력:
   ```
   TELEGRAM_BOT_TOKEN=5번에서_받은_토큰
   TELEGRAM_CHAT_ID=6번에서_확인한_채팅ID
   ```

---

## STEP 7. 안드로이드 앱 설치

1. GitHub 저장소 → Actions 탭 클릭
2. **Flutter APK 자동 빌드** 워크플로우 선택
3. 가장 최근 실행 클릭 → Artifacts 섹션에서 APK 다운로드
4. 파일을 스마트폰으로 전송 (카카오톡, 메일 등)
5. 스마트폰에서 APK 파일 탭 → 설치
   - 설치 전 설정 → 출처를 알 수 없는 앱 허용 필요

---

## 자주 묻는 질문

**Q. 실행했는데 오류가 나요**
→ .env 파일에 입력한 값이 맞는지 확인하세요

**Q. 포트가 이미 사용 중이라고 해요**
→ Docker Desktop에서 실행 중인 컨테이너를 모두 중지 후 다시 시도

**Q. 데이터가 사라졌어요**
→ `docker-compose down -v` 는 데이터도 삭제됩니다. 중지할 때는 반드시 `docker-compose down` 만 사용하세요
