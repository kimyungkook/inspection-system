1. 시스템 개요

검사기록관리 시스템 v01은 생산된 제품의 바코드를 스캔하여 등록된 정보와 일치하는지 자동으로 검수하는 시스템입니다.

바코드 스캔

USB 스캐너 또는 카메라로 바코드를 스캔합니다

자동 검수

등록된 정보와 자동으로 비교하여 PASS/FAIL 판정

기록 관리

모든 검사 기록을 저장하고 Excel로 내보내기 가능

시스템명검사기록관리 시스템
버전v01
지원 환경PC (윈도우), 태블릿, 스마트폰 (웹 브라우저)
기본 계정admin / admin1234 (관리자), inspector01 / inspector1234 (검사원)

2. 로그인 방법

1
웹 브라우저를 열고 시스템 주소를 입력합니다. (예: http://서버IP:3000)
2
로그인 화면에서 아이디와 비밀번호를 입력합니다.
3
로그인 버튼을 클릭하거나 Enter를 누릅니다.
Tip: 최초 접속 시 관리자(admin)로 로그인하여 사용자를 추가하세요. 비밀번호는 로그인 후 즉시 변경하는 것을 권장합니다.

4. 제품 등록 방법

검사 전에 검사할 제품의 정보를 먼저 시스템에 등록해야 합니다.

1
왼쪽 메뉴에서 제품 등록을 클릭합니다.
2
오른쪽 상단 새 제품 등록 버튼을 클릭합니다.
3
필수 항목을 입력합니다:
  • 자재코드 (AP CODE): 라벨의 AP CODE 또는 자재코드 (예: B3224354)
  • 모델명 (품명): 제품명 (예: FBGA 8.2X12.4 Rotation_JUMBO DANPLA)
  • LOT NO: 생산 LOT 번호 (예: 20260113001)
4
바코드 스캔값을 입력합니다 (USB 스캐너로 각 필드에 직접 스캔 가능):
  • 긴 바코드 값: LOT NO 바코드를 스캔한 값
  • 짧은 바코드 값: AP CODE 바코드를 스캔한 값
  • 사이즈 바코드 값: 사이즈 라벨 바코드 스캔값
5
저장 버튼을 클릭하면 등록 완료됩니다.
Tip: 바코드 필드에 커서를 놓고 USB 스캐너로 직접 스캔하면 자동으로 입력됩니다.

5. 바코드 검사 방법 (핵심)

A. USB 바코드 스캐너 사용 방법
1
왼쪽 메뉴에서 바코드 검사를 클릭합니다.
2
왼쪽 패널의 검사 제품 정보를 입력합니다.
또는 "등록된 제품 불러오기" 버튼으로 기존 제품을 선택합니다.
3
검사 세션 시작 버튼을 클릭합니다.
4
스캔 항목 버튼을 선택합니다:
LOT NO (긴) → 소형 라벨의 LOT NO 바코드
자재코드 (짧은) → 대형 라벨의 AP CODE 바코드
사이즈 → 사이즈 라벨 바코드
5
중요! 화면 중앙의 바코드 입력 칸을 마우스로 클릭하여 포커스를 맞춥니다.
USB 스캐너는 키보드처럼 동작합니다. 반드시 입력 칸이 선택(파란 테두리)된 상태에서 스캔하세요!
6
제품 라벨의 바코드에 스캐너를 가져다 대고 스캔합니다. 자동으로 결과가 표시됩니다.
7
모든 수량 검사 완료 후 검사완료 버튼을 클릭합니다.
B. 카메라 스캔 사용 방법 (태블릿/모바일)
1
스캔 영역 오른쪽 상단의 카메라 버튼을 클릭합니다.
2
카메라 접근 허용 팝업이 뜨면 허용을 선택합니다.
3
화면의 사각형 가이드 안에 바코드를 위치시키면 자동으로 스캔됩니다.
Tip: 카메라 스캔 시 바코드를 가이드 박스 중앙에 맞추고, 흔들리지 않게 잠시 고정하세요. 조명이 충분한 환경에서 스캔하면 인식률이 높아집니다.

6. 검사 결과 확인 (PASS / FAIL)

PASS

스캔값 = 등록값

화면이 초록색으로 표시됩니다

FAIL

스캔값 ≠ 등록값

화면이 빨간색으로 표시됩니다

상황조치 방법
PASS 표시 정상입니다. 다음 제품을 스캔하세요.
FAIL 표시 - 다른 LOT 제품 해당 제품을 별도로 구분하고, 관리자에게 보고하세요.
FAIL 표시 - 스캔 오류 입력 칸에 포커스가 있는지 확인하고 다시 스캔하세요.
등록값 없음 오류 검사 제품 정보에 해당 바코드 값이 입력되었는지 확인하세요.

7. 검사 기록 조회 및 Excel 내보내기

기록 검색
  • LOT NO로 검색: LOT NO 필드에 입력
  • 자재코드로 검색: 자재코드 필드에 입력
  • 날짜로 검색: 시작일/종료일 입력
  • PASS/FAIL 필터: 결과 드롭다운 선택
Excel 내보내기

검사 기록 페이지 오른쪽 상단의 Excel 내보내기 버튼을 클릭하면 현재 검색 조건의 기록이 Excel 파일로 다운로드됩니다.

8. 자주 묻는 질문 (FAQ)

바코드 입력 칸에 커서가 위치해야 합니다. 화면의 바코드 입력 칸을 마우스로 한 번 클릭한 후 스캔하세요. 입력 칸에 파란 테두리가 표시되면 준비된 상태입니다.

다음을 확인하세요:
  1. 스캔 항목 버튼이 올바르게 선택되어 있는지 확인 (LOT NO 스캔 시 "LOT NO(긴)" 선택)
  2. 등록된 제품 정보의 바코드 값이 정확한지 확인
  3. 스캔된 값과 등록값을 화면에서 직접 비교

네, 가능합니다. 시스템은 네트워크 기반으로 동작하므로 여러 PC, 태블릿에서 동시 접속이 가능합니다. 각자의 계정으로 로그인하면 됩니다.

관리자 또는 담당자 계정으로 검사 기록 페이지에서 해당 기록을 삭제할 수 있습니다. 검사원 계정은 삭제 권한이 없으므로 관리자에게 요청하세요.

아닙니다. 초기화 버튼은 화면의 입력 내용만 지웁니다. 데이터베이스에 저장된 검사 기록은 그대로 보존됩니다.
